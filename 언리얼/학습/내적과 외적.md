## 내적(Dot Product)
두 벡터의 내적은 다음과 같이 정의된다.

$$
a · b = |a| |b| cos θ
$$

- `a`, `b`: 두 벡터
- `|a|`, `|b|`: 각 벡터의 크기
- `θ`: 두 벡터 사이의 각도

### 특징

- 결과값은 **스칼라**이다. (크기만 가지고 방향은 없는 값)
- 두 벡터가 **같은 방향**일수록 내적 값은 커진다.
- 두 벡터가 **수직**일 때 내적 값은 0 이다.
- 두 벡터가 **반대 방향**일수록 내적 값은 작아진다.


> [!Example] 예시
> 두 벡터가 완전히 반대 방향일 때 내적 값은 최소값을 갖습니다. 좀 더 자세히 설명드리겠습니다.
> 
> 내적은 두 벡터의 크기와 두 벡터 사이의 각도의 코사인 값을 곱한 결과입니다. 즉, 두 벡터가 이루는 각이 0도일 때 코사인 값이 1로 최대가 되고, 180도일 때 코사인 값이 -1로 최소가 됩니다.
>
> 따라서 두 벡터가 완전히 반대 방향일 때, 즉 두 벡터 사이의 각도가 180도일 때 내적 값은 최소값을 갖게 됩니다. 이때 내적 값은 두 벡터 크기의 곱에 -1을 곱한 값과 같습니다.
>
> 예를 들어, 크기가 3인 벡터 A와 크기가 2인 벡터 B가 완전히 반대 방향이라고 가정해 보겠습니다. 이 경우 두 벡터의 내적은 3 x 2 x -1 = -6이 됩니다.


### 활용

- **두 벡터 사이의 각도 계산:** 위 공식을 변형하여 $cos θ = (a · b) / (|a| |b|)$ 를 이용하여 각도를 구할 수 있다.
- **투영 :** 한 벡터를 다른 벡터에 투영한 벡터의 길이를 구할 수 있다.
- **조명 계산 :** 빛 벡터와 표면 법선 벡터의 내적을 통해 표면의 밝기를 계산한다.
- **전 후방 계산 :** 적 객체가 플레이어 앞에 있는지 뒤에있는지 판별할 수 있다.

```C++
// 적 캐릭터의 방향을 계산합니다.
FVector DirectionVector = EnemyActorLocation - GetActorLocation();

// 캐릭터의 forward 벡터와 히트 적 객체 사이의 각도를 계산합니다. 
float DotProduct = FVector::DotProduct(GetActorForwardVector(), DirectionVector); 
float Angle = FMath::Acos(DotProduct); 

// 각도를 도 단위로 변환합니다. 
Angle = FMath::RadiansToDegrees(Angle);

// Angle 이 45.0f 보다 작다면 정면, 135.0f 보다 크면 후면입니다.
```


## 외적(Cross Product)
두 벡터 **a**와 **b**의 외적은 다음과 같이 정의된다.

$$
a × b = |a| |b| sin θ n
$$

- `a`, `b`: 두 벡터
- `|a|`, `|b|`: 각 벡터의 크기
- `θ`: 두 벡터 사이의 각도 (0° ≤ θ ≤ 180°)
- `n`: 두 벡터에 모두 수직인 단위 벡터 (오른손 법칙을 따름)

### 기하학적 의미

외적의 크기는 두 벡터가 이루는 평행사변형의 넓이와 같다.
외적 벡터의 방향은 두 벡터가 이루는 평면에 수직이며, 오른손 법칙을 사용하여 결정된다.

### 오른손 법칙

오른손 법칙은 외적 벡터의 방향을 결정하는 규칙이다.

- 오른손의 네 손가락을 첫 번째 벡터 **a** 방향에서 두 번째 벡터 **b** 방향으로 감싸 쥐었을 때, 엄지손가락이 가리키는 방향이 외적 벡터 **a × b**의 방향이다.

### 성질

- **반대칭성:** a × b = - (b × a)
- **선형성:** a × (b + c) = (a × b) + (a × c)
- **자기 외적:** a × a = 0
- **두 벡터가 평행:** a × b = 0 (a와 b가 평행하면 θ = 0° 또는 180° 이므로 sin θ = 0)
- **두 벡터가 수직:** |a × b| = |a| |b| (a와 b가 수직이면 θ = 90° 이므로 sin θ = 1)

### 활용

외적은 게임 개발 및 컴퓨터 그래픽스 분야에서 다양하게 활용된다.

- **법선 벡터 계산:** 평면 위의 두 벡터의 외적을 계산하면 평면에 수직인 법선 벡터를 얻을 수 있다.
- **회전력 (토크) 계산:** 힘 벡터와 작용점까지의 위치 벡터의 외적을 통해 회전력 (토크)을 계산할 수 있다.
- **두 벡터가 이루는 평면 판별:** 두 벡터의 외적이 0 벡터이면 두 벡터는 평행하며, 같은 평면에 있다.
- **3차원 공간에서의 거리 계산:** 점과 직선 사이의 거리, 두 직선 사이의 거리 등을 계산할 때 외적을 활용할 수 있다.

추가적으로 언리얼 엔진에서는 `FVector::CrossProduct` 함수를 사용하여 두 벡터의 외적을 계산할 수 있다.

```C++
FVector a = FVector(1, 0, 0);
FVector b = FVector(0, 1, 0);
FVector c = FVector::CrossProduct(a, b); // c = (0, 0, 1)
```
