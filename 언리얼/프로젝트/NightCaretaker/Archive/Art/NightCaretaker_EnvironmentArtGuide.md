# NightCaretaker Environment Art Guide

## 문서 정보

- 문서 종류: 환경 아트 제작 가이드
- 버전: v0.1
- 작성일: 2026-03-22
- 대상: Environment Artist, Outsource Artist

## 1. 목적

이 문서는 본 프로젝트의 환경 아트를 제작할 때 필요한 공간 언어, 자산 재사용 원칙, Nanite 적용 기준, 우선 제작 순서를 정리한 실무용 가이드다.

## 2. 전체 제작 철학

- 하나의 건물을 깊게 만든다.
- 많은 공간을 만드는 것보다 반복 방문해도 질리지 않는 변주를 만든다.
- 실사적인 공간 설득력은 구조, 표면, 생활 흔적에서 나온다.
- 초현실은 새로운 판타지 공간을 만드는 대신, 기존 공간의 규칙을 어긋나게 만드는 방식으로 표현한다.

## 3. 핵심 제작 우선순위

1. 복도 모듈 세트
2. 현관문, 문패, 우편함, 층 표기, 표지판
3. 형광등, 비상등, 스위치, 배전반, 전기 계량기
4. 관리실 세트
5. 지하 전기실 세트
6. 계단실 / 엘리베이터 앞 세트
7. 세대 내부 기본 가구 세트
8. 307호 전용 변질 세트

## 4. Nanite 적용 기준

- 벽체, 천장 구조물, 바닥 구조물, 배관, 덕트, 분전함, 난간, 엘리베이터 프레임은 Nanite 우선
- 반복 사용되는 배경 소품도 가능하면 Nanite 우선
- 표면 밀도가 중요한 자산은 노멀맵 과장보다 실제 형상 밀도를 우선
- 플레이어가 손으로 직접 조작하는 액터는 기능 요구에 따라 예외 처리 가능
- Nanite 적용 여부보다 중요한 것은 "가까이서 봤을 때 구조가 싸 보이지 않는가"이다

## 5. 모듈화 원칙

- 복도는 길이별 2~3종 모듈로 분해
- 현관문은 기본형 1개에 번호판, 체인, 도어락, 손상 상태로 변주
- 벽은 페인트 상태, 얼룩, 습기, 균열, 보수 흔적으로 변주
- 천장 조명은 동일한 fixture를 상태별로 재활용
- 세대 내부는 구조보다 소품 상태와 조명 상태로 차이를 만든다

## 6. 표면 언어

- 벽: 오래된 도장면, 생활 얼룩, 부분 보수 자국
- 바닥: 장판, 타일, 콘크리트, 습기, 먼지 축적
- 금속: 손때, 산화, 칠 벗겨짐, 유지보수 흔적
- 전기 설비: 기능적이고 현실적인 라벨, 경고 스티커, 케이블 정리 상태 차이
- 목재/가구: 값싼 합판 느낌, 사용감, 미세한 뒤틀림

## 7. 공간별 제작 메모

### 복도

- 게임의 대표 공간이므로 가장 먼저 정리
- 길이감보다 반복 방문 시 변주가 가능한 구조가 중요
- 문 수, 우편함, 표지판, 소화기, CCTV 더미, 게시물, 형광등 간격이 리듬을 만든다
- 끝이 완전히 보이기보다, 보이지만 확신이 흔들리는 구성이 좋다

### 관리실

- 플레이어의 기준점 역할
- 너무 따뜻하거나 아늑하지 말 것
- 책상, 서류, 민원 보드, 형광등, 낡은 수납장, 작은 생활 물품으로 현실성을 만든다

### 계단실 / 엘리베이터 앞

- 이동 연결 공간이지만 감정 압력이 있어야 함
- 난간, 층수 표기, 비상등, 콘크리트 질감, 소음이 반사될 법한 구조가 중요

### 세대 내부

- 기본형 4개 정도를 염두에 두고 설계
- 차이는 가족 구성, 정리 상태, 조명 상태, 생활 흔적, 이상 징후로 만든다
- 가구 수를 늘리기보다 공간 성격이 빠르게 읽히게 할 것

### 지하 전기실

- 습기와 설비 압박이 핵심
- 배관, 밸브, 배전반, 전선관, 응급 조명, 경고 표지판이 기능적으로 보여야 함
- 공포는 고어가 아니라 "여기 오래 머물고 싶지 않다"는 감각에서 나와야 함

### 307호

- 초기에는 일반 세대처럼 보여야 함
- 후반으로 갈수록 번호, 문, 복도 연결감, 깊이감, 반사, 암부가 틀어짐
- 별도 판타지 오브젝트를 많이 추가하지 말고 기존 세대 구조를 비정상적으로 변주

## 8. 변주 규칙

- 같은 복도를 다시 사용할 때 최소 하나는 바꾼다
- 문 번호 또는 표기 정보 불일치
- 조명 상태 변경
- 문 열림 / 닫힘 상태 변경
- 우편함 상태 변경
- 생활 흔적 추가 또는 제거
- 소품 배치의 작은 어긋남

## 9. 금지 사항

- 한 번만 보고 지나갈 과도한 고유 자산 양산
- 의미 없는 폐허 스타일 과장
- 현실 생활 공간으로 보이지 않는 럭셔리 인테리어
- 괴물 비주얼을 전제로 한 파손/혈흔 중심 연출
- 플레이어 경로를 읽기 어렵게 만드는 과한 장식

## 10. 1차 산출물 체크리스트

- 복도 기본형 key image
- 복도 모듈 시트
- 현관문 / 번호판 / 우편함 변주안
- 관리실 key image
- 지하 전기실 key image
- 세대 내부 기본형 key image
- 307호 변질 단계 3안
- 반복 변주 예시 before / after 2세트

## 11. 리뷰 질문

- 같은 자산을 재사용해도 다른 공간처럼 읽히는가
- 가까이 붙었을 때 자산 밀도가 싸 보이지 않는가
- 플레이어 동선과 상호작용 포인트가 명확한가
- 빛이 바뀌면 공간 감정도 함께 바뀌는 구조인가
- 307호 변주는 갑작스러운 판타지가 아니라 누적된 어긋남으로 느껴지는가
