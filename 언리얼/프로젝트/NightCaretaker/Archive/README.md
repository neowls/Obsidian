# NightCaretaker Archive

## 목적

이 폴더는 통합 문서로 흡수된 기존 상세 문서 원본을 보관한다.
현재 활성 판단 기준은 `Document` 루트의 통합 문서 3개다.

## 구성

- `Planning/`: GDD, 게임플레이 가이드, 공포 가이드, 307 비트시트, 민원 팩 원본
- `Development/`: 개발 기획, 런타임 프레임워크, RealityCam, 물리 상호작용, 데이터 작성/매핑 원본
- `Art/`: 아트 브리프, 심리 공포 아트, 환경 가이드, 라이팅/사운드 무드, 에셋 우선순위, 제작 티켓 원본

## 사용 원칙

- 기본적으로는 이 폴더를 먼저 읽지 않는다.
- 통합 문서의 기준이 어떻게 생겼는지 추적하거나 원문 표현을 확인할 때만 연다.
