## 개요
1인칭, 3인칭 메쉬 각각의 애니메이션 블루프린트(Animation Blueprint)를 활용하여 애님을 구현한다.

## 구현
![[Pasted image 20250210120631.png]]

기본적으로 제공되는 ABP 를 통해 각 움직임에 대한 애니메이션을 구동해 보았으며, 이를 기반으로 GroundSpeed 와 Direction 에 따른 ABP를 따로 구성하였다.

![[Pasted image 20250210120642.png]]

![[Pasted image 20250210120650.png]]

Mannequin Arm 메쉬는 별도로 구할 수 없어 블렌더를 활용하여 팔을 제외한 나머지 폴리곤들을 제거하여 생성해주었다.
이때 블렌더에서 Export 할 때 최상단 그룹을 제외한 포즈와 root, LOD 를 선택하고 Select Export 옵션을 체크해야만 정상적으로 언리얼에서 Import 할 수 있다.

```C++ Header
// Fill out your copyright notice in the Description page of Project Settings.

#pragma once

#include "CoreMinimal.h"
#include "Animation/AnimInstance.h"
#include "TACAnimInstance.generated.h"

UCLASS()
class TAC13_API UTACAnimInstance : public UAnimInstance
{
	GENERATED_BODY()

public:
	UTACAnimInstance();
	FORCEINLINE class ATACCharacterPlayer* GetCharacter() const { return Character; }

	virtual void NativeBeginPlay() override;
	virtual void NativeUpdateAnimation(float DeltaSeconds) override;

	void SetMovement();
	//void InterpAiming();
```

```C++
// Fill out your copyright notice in the Description page of Project Settings.

#include "TACAnimInstance.h"

#include "KismetAnimationLibrary.h"
#include "Character/TACCharacterPlayer.h"
#include "GameFramework/CharacterMovementComponent.h"

UTACAnimInstance::UTACAnimInstance()
{
	bIsMoving = false;
	bIsSprint = false;
	bIsJumping = false;
	bIsFalling = false;
	bIsAiming = false;
	bInterpAiming = false;
}

void UTACAnimInstance::NativeBeginPlay()
{
	Super::NativeBeginPlay();
	Character = CastChecked<ATACCharacterPlayer>(TryGetPawnOwner());
	if(!Character)
	{
		UE_LOG(LogTemp, Warning, TEXT("Failed to Get Character"));
	}
}

void UTACAnimInstance::NativeUpdateAnimation(float DeltaSeconds)
{
	Super::NativeUpdateAnimation(DeltaSeconds);
}

void UTACAnimInstance::SetMovement()
{
	Velocity = Character->GetVelocity();
	GroundSpeed = Velocity.Size2D();
	bIsMoving = GroundSpeed > 1.f;
	bIsSprint = false;
	Direction = UKismetAnimationLibrary::CalculateDirection(Character->GetVelocity(), Character->GetActorRotation());
	bIsFalling = Character->GetCharacterMovement()->IsFalling();
	bIsJumping = bIsFalling & (Velocity.Z > 100.f);
	AimPitch = Character->GetBaseAimRotation().GetNormalized().Pitch;
}
```

기본적인 상태 변수를 지정하였고, 이를 통해서 애니메이션 블루프린트가 해당 조건을 활용할 수 있게 하였다.
상·하체 분리를 통해서 상반신은 플레이어가 조준하고 있는 방향에 따라 AimOffset 을 적용하였고, 하체는 액터의 방향 연산을 통해 별도로 애니메이션이 진행되게 하였다.
