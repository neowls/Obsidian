## 개요
적이 사망했을 때 점차 사라지는 디졸브 효과를 적용한다.

## 구현
기본적으로 디졸브 효과는 머터리얼의 마스킹을 통해 구현된다.

![[Dissolve_1.png]]

기존 텍스쳐에 특정 노이즈나 패턴 텍스쳐에서 계산된 값을 오퍼시티 마스크값에 블렌딩하여 구현되는 것이 가장 일반적이다.
추가적으로 Emissive 노드에 디졸브 테두리 부분을 특정 컬러로 가산하여 타들어가는 듯한 느낌을 줄 수 있다.
### 사용된 노드들
- Saturate : 입력 값을 0과 1사이로 제한(Clamp)하는 함수
- Subtract : A-B 뺄셈 연산을 하는 함수 (MaskTexture 에서 특정 Amount 값을 뺀다면, Amount 값이 증가될 수록 더 많이 제거됨)
- SmoothStep : Min, Max, Value 로서 보간(Lerp) 함수로서 특정 범위내에서 값을 점진적으로 변화시키고 경계를 부드럽게 해줌. (디졸브 경계면을 흐리게 하는 역할)
