## 개요
무기 테이블을 통해 플레이어가 획득한 무기를 착용할 때 알맞은 애니메이션 레이어와 메쉬를 적용한다.

## 구현

```C++
//KYStruct.h
// 무기 타입  
UENUM(BlueprintType)  
enum class EWeaponType : uint8  
{  
    None,  
    Sword,  
    GreatSword,  
    DualBlade,  
    Scythe,  
    Hammer,  
    Spear  
};  
  
// 무기 데이터  
USTRUCT(BlueprintType)  
struct FWeaponData : public FTableRowBase  
{  
    GENERATED_BODY()  
  
public:  
    FWeaponData()  
       : Mesh(nullptr)  
       , AttackSpeed(1.0f)  
       , Damage(0.0f)  
       , Type(EWeaponType::None) {}  
  
    UPROPERTY(EditDefaultsOnly, BlueprintReadWrite, Category="Weapon Info")  
    TObjectPtr<class UStaticMesh> Mesh;  
      
    UPROPERTY(EditDefaultsOnly, BlueprintReadWrite, Category="Weapon Info")  
    float AttackSpeed;  
  
    UPROPERTY(EditDefaultsOnly, BlueprintReadWrite, Category="Weapon Info")  
    float Damage;  
  
    UPROPERTY(EditDefaultsOnly, BlueprintReadWrite, Category="Weapon Info")  
    EWeaponType Type;  
};
```

우선 데이터 테이블로 만들 수 있도록 구조체를 선언해주었다.
무기의 메쉬와 공격속도, 공격력, 타입 등을 가진다.

```C++
//KYGameSingleton.h

UCLASS()  
class PROJECTKY_API UKYGameSingleton : public UObject  
{  
    GENERATED_BODY()  
  
public:  
    UKYGameSingleton();  
      
    static UKYGameSingleton& Get();  
  
    UFUNCTION(blueprintCallable, Category = "Singleton")  
    static UKYGameSingleton* GetSingleton();  
      
    UFUNCTION(BlueprintCallable,  Category = "Ability System")  
    FWeaponData GetWeaponDataByRowName(const FName& InName) const;  
  
private:  
    UPROPERTY()  
    TObjectPtr<UDataTable> WeaponDataTable;  
  
};
```

```C++
//KYGameSingleton.cpp

  
UKYGameSingleton::UKYGameSingleton()  
{  
    static ConstructorHelpers::FObjectFinder<UDataTable> WeaponDataTableRef(TEXT("/Game/_Dev/DataTable/DT_MeleeWeapon.DT_MeleeWeapon"));  
    if (WeaponDataTableRef.Object != nullptr)  
    {  
       WeaponDataTable = WeaponDataTableRef.Object;  
       check(WeaponDataTable->GetRowMap().Num() > 0);  
    }  
}  
  
UKYGameSingleton& UKYGameSingleton::Get()  
{  
    UKYGameSingleton* Singleton = CastChecked<UKYGameSingleton>(GEngine->GameSingleton);  
    if (Singleton)  
    {  
       return *Singleton;  
    }  
  
    KY_LOG(LogKY, Error, TEXT("Invalid Game Singleton"));  
    return *NewObject<UKYGameSingleton>();  
}  
  
UKYGameSingleton* UKYGameSingleton::GetSingleton()  
{  
    UKYGameSingleton* Singleton = CastChecked<UKYGameSingleton>(GEngine->GameSingleton);  
    if (Singleton)  
    {  
       return Singleton;  
    }  
  
    KY_LOG(LogKY, Error, TEXT("Invalid Game Singleton"));  
    return NewObject<UKYGameSingleton>();  
}  
  
FWeaponData UKYGameSingleton::GetWeaponDataByRowName(const FName& InName) const  
{  
    static const FString ContextString(TEXT("Weapon Data Table Context"));  
    FWeaponData* RowData = WeaponDataTable->FindRow<FWeaponData>(InName, ContextString);;  
    if (!RowData)  
    {  
       KY_LOG(LogKY, Warning, TEXT("Can't Find Weapon With This Name : %s"), *InName.ToString());  
       return FWeaponData();  
    }  
    return *RowData;  
}
```

데이터 테이블을 관리하기 위해 싱글톤을 생성해주었으며, 이후 언리얼에서 자체적으로 제공하는 싱글톤 설정을 통해 등록하였다.
기본적으로 데이터 테이블에서 FName 통해 해당하는 무기의 행을 찾아 데이터를 반환한다.
또한 FindRow 로직은 내부 맵에서 키값을 통해 찾기 때문에 빠른 접근이 가능하다.

![[Weapon_Datatable_1.png]]

아이템에서는 싱글톤을 통해 현재 객체에 지정된 Weapon FName 통해서 Weapon Data 구조체를 가져오고, 이를 저장하여 이후 캐릭터가 획득하였을때 해당 정보를 넘겨준다.

```C++
void AKYCharacterPlayer::AddWeaponData(const FName& WeaponName, const FWeaponData& InWeaponData)  
{  
    Super::AddWeaponData(WeaponName, InWeaponData);  
    KY_LOG(LogKY, Log, TEXT("Equipped Weapon Num : %d"), EquippedWeapon.Num());  
    if (EquippedWeapon.Num() < 3)  
    {  
       EquippedWeapon.Add(WeaponName);  
    }  
}
```

현재 장착된 무기 정보는 `TArray<FName>` 형태로 저장되고, 소유하고 있는 무기에 대한 정보는 `TMap<FName, FWeaponData>` 형태로 저장된다.

```C++
void AKYCharacterPlayer::ChangeWeaponWithType(const FInputActionValue& Value)  
{  
    if(ASC->HasMatchingGameplayTag(UKYGameplayTags::CharacterState.IsCombat)) return;  
    float AxisValue = Value.Get<float>();  
    int32 WeaponIndex = FMath::RoundToInt(AxisValue);  
      
    if(EquippedWeapon.IsValidIndex(WeaponIndex - 1))  
    {  
       ChangeWeapon(EquippedWeapon[WeaponIndex - 1]);  
    }  
}

void AKYCharacterBase::ChangeWeapon(FName WeaponName)  
{  
    if (!OwnWeaponData.Contains(WeaponName))  
    {  
       KY_LOG(LogKY, Warning, TEXT("Can't find %s Weapon"), *WeaponName.ToString());  
       return;  
    }  
    UpdateWeaponData(OwnWeaponData.FindRef(WeaponName));  
}
```

키 입력에 따라 해당하는 인덱스의 무기 FName 을 전달하며, 해당되는 무기를 가지고 있으면 소유하고 있는 무기 맵에서 무기 정보를 가져와 업데이트 한다.
