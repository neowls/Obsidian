## 개요
아이템 획득 시 나타나는 UI를 구현한다.

## 구현
우선 기존의 ASC 를 설정해주는 커스텀 UserWidget 을 상속받아 제작하였다.

![[GetItemUI_0.png]]

우선 이미지와 텍스트 블록으로 구성하였으며, 현재 가진 골드를 나타내는 텍스트는 바인딩으로 구현하였다.

![[GetItemUI_1.png]]

현재 ASC가 가진 어트리뷰트가 변경된다면 이벤트를 보내고 Lerp 함수를 통해 텍스트가 점차적으로 변경되는 효과를 구현하였다.

![[GetItemUI_2.png]]

이벤트를 수신한다면 오퍼시티를 변경하는 페이드 인 애니메이션을 재생해주고, 일정 시간이 지나면 다시 사라지도록 페이드 아웃 애니메이션을 재생하게 해주었다.
