## 개요
체간(혹은 그로기)라고도 부르며, 값에 따라 각기 다른 모션을 취할 수 있도록 어트리뷰트 세트로 구현하였다.
어트리뷰트로는 스탠스(Stance), 최대 스탠스(Max Stance), 최소 스태거 스탠스(Min Stagger Stance), 최대 스태거 스탠스(Max Stagger Stance) 메타데이터로서 사용되는 인커밍 스탠스(Incoming Stance)로 구분하였다.

## 정리
피격 반응 시스템인 스탠스는 수치에 따라 피격 상태 유발을 구현한다.
몬스터들은 각자 다른 스탠스 수치를 가지고 있으며 이에 따라 구현의 폭을 넓힐 수 있다.

캐릭터의 공격은 약공격, 강공격, 어퍼, 슬램, 블라스트 공격에 대한 부분으로 나뉜다.
지정한 스탠스 수치를 넘겨서 공격받으면 해당 공격에 따라 넉백될지 에어본될지 슬램될지 지정할 수 있다.

- 최소 스태거 스탠스를 넘기는 공격을 받으면 스태거 상태가 되며 스태거 액션을 취한다.
- 현재 스탠스 값이 최대 스태거 상태를 넘긴 상태라면 탈진 상태가 되며, 이때는 어떠한 공격이더라도 스태거 액션을 취한다.
- 스탠스 값이 최대 스탠스 값에 도달하면 넉아웃 상태가 된다.
- 넉아웃 상태에서 공중에 뜬 상태라면 스탠스 값이 줄어들지 않는다.
- 넉아웃 상태에서 바닥에 닿았을 시 넉다운 상태로 전환되며, 스탠스 수치를 초기화한다.

각 상태별 구간의 갭을 얼마나 넓히느냐에 따라 몬스터의 피격 액션이 정해지기에 난이도를 조절할 수 있다.
또한 강한 공격을 받았을 경우 그만큼 많은 스탠스 값이 들어와 곧바로 넉다운 상태가 될 수 있다.

0 - 통상 상태 피격 액션이 없다.
1 - 스태거 상태(Staggered)
2 - 넉아웃 상태(Knock Out)

## 스탠스 상태
스태거(Staggered) - 멈칫하는 상태로 행동을 할 수 없으며 짧은 시간내에 풀린다.
녹다운(KnockDown) - 쓰러진 상태로 행동을 할 수 없으며, 바닥에 닿아야만 일정 시간지나고 일어날 수 있다.

만약 녹다운이 되어있는 상태라면 스태거 상태로 전환하지 않는다.
하지만 스태거 상태라면 녹다운 상태로 전환한다.
항상 녹다운이 스태거보다 우선순위가 높다. (스태거 < 녹다운)

플레이어는 Light, Heavy, Upper, Slam, Blast 의 공격을 가지고 있다.

각 **모먼텀 수치를 프로퍼티로 두어 조절**할 수 있게한다.
이는 각 캐릭터마다 태그와 정해진 모먼텀 수치를 데이터 에셋으로 등록한다.
Damage GA는 이러한 데이터를 어빌리티를 등록할 때 가져오고, 피격 이벤트가 발생했을 때 스탠스 수치에 따라 해당하는 피격 상태와 스탠스 상태를 지정한다.

    UPROPERTY(BlueprintReadWrite, Category="Attributes", Meta=(AllowPrivateAccess=true))

```C++
UKYAttributeSetStance::UKYAttributeSetStance() :  
    Stance(0.f),  
    MaxStance(300.0f),  
    StaggerStance(100.0f),  
    KnockDownStance(200.0f)  
{  
}  
  
void UKYAttributeSetStance::ClampAttributeOnChange(const FGameplayAttribute& Attribute, float& NewValue) const  
{  
    if (Attribute == GetStanceAttribute())  
    {  
       NewValue = FMath::Clamp(NewValue, 0.0f, GetMaxStance());  
    }  
}  
  
void UKYAttributeSetStance::PostGameplayEffectExecute(const struct FGameplayEffectModCallbackData& Data)  
{  
    Super::PostGameplayEffectExecute(Data);  
    if (Data.EvaluatedData.Attribute == GetIncomingStanceAttribute())  
    {  
       float IncomingStanceDone = GetIncomingStance();  
       SetIncomingStance(0.0f);  
       if(IncomingStanceDone > 0.0f)  
       {  
          const float NewStance = GetStance() + IncomingStanceDone;  
          SetStance(NewStance);  
            
          if (OnStanceChange.IsBound())  
          {  
             const FGameplayEffectContextHandle& EffectContextHandle = Data.EffectSpec.GetEffectContext();  
             AActor* Causer = EffectContextHandle.GetEffectCauser();  
  
             OnStanceChange.Broadcast(Causer, Data.EffectSpec.CapturedSourceTags.GetSpecTags(), GetCurrentStanceState());  
          }  
       }  
    }  
}  
  
uint8 UKYAttributeSetStance::GetCurrentStanceState() const  
{  
    return GetStance() >= GetKnockDownStance() ? 2 : GetStance() >= GetStaggerStance() ? 1 : 0;  
}
```

피격시 GE를 통해서 스탠스 값이 전달되며, 이는 공격마다 다른 값을 가지게 할 수 있다.
우선 스탠스는 총 3단계로 일반, 스태거, 넉다운 세 개로 구분된다.
현재 스탠스 값이 지정된 스태거나 넉다운 스탠스보다 높아졌다면 이러한 상태를 전달하는 것이다.
또한 스탠스가 변경되었다는 델리게이트를 바인딩한 객체가 있다면 이러한 현재 스탠스 상태값을 전달한다.
