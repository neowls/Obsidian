## 개요
특정 코스트를 사용하며, 범위 내에 있는 적들에게 데미지를 주는 스킬을 구현한다.

## 구현

![[AOS_Attack_Property_0.png]]

어빌리티 자체는 베이스 어빌리티를 상속받아 쉽게 만들 수 있었으며, 몽타주와 데이터 태그 및 이벤트 발생만 작성하였다.

![[AOS_Attack_Montage_0.png]]

콜백 함수에 대한 발동은 노티파이로 SendGameplayEvent 를 보내주게 하였다.

```C++
#pragma once  
  
#include "CoreMinimal.h"  
#include "GAS/GameAbility/KYGameplayAbility.h"  
#include "Struct/KYStruct.h"  
#include "KYGA_AnimBase.generated.h"  
  
/**  
 * */  
UCLASS()  
class PROJECTKY_API UKYGA_AnimBase : public UKYGameplayAbility  
{  
    GENERATED_BODY()  
  
public:  
    UKYGA_AnimBase();  
  
protected:  
    virtual void ActivateAbility(const FGameplayAbilitySpecHandle Handle, const FGameplayAbilityActorInfo* ActorInfo, const FGameplayAbilityActivationInfo ActivationInfo, const FGameplayEventData* TriggerEventData) override;  
  
    UFUNCTION(BlueprintNativeEvent)  
    void OnSimpleEventReceivedCallback(FGameplayEventData Payload);
```
![[AOS_Attack_Node_0.png]]

또한 이벤트 콜백 함수에 BluepirntNativeEvent 지정자를 두어 블루프린트에서도 어빌리티의 이벤트 콜백에 대한 로직을 추가할 수 있게 하였다.
스피어 오버랩을 통해 범위내에 있는 모든 적들에게 지정된 게임플레이 이펙트를 적용하게 구현하였다.

![[AOS_Attack_Property_1.png]]

코스트는 어빌리티에서 GameplayEffect 통해 쉽게 적용할 수 있으며, 얼마나 사용될지에 대한 부분을 Scalar 값으로 지정할 수 있다.
스킬의 레벨이나 특정 상태에 따른 레벨링 커브 테이블을 통해 코스트 양을 조절할 수 있다.
