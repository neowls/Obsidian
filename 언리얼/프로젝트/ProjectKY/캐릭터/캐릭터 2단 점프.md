## 개요
기존의 점프 어빌리티를 상속받아 구현한다.

## 구현

```C++
UPROPERTY(EditDefaultsOnly, BlueprintReadWrite, Category="Jump", meta=(AllowPrivateAccess=true))  
int32 JumpMaxCount = 1;
```

```C++
void UKYGA_Jump::OnGiveAbility(const FGameplayAbilityActorInfo* ActorInfo, const FGameplayAbilitySpec& Spec)  
{  
    Super::OnGiveAbility(ActorInfo, Spec);  
    Cast<ACharacter>(ActorInfo->AvatarActor)->JumpMaxCount = JumpMaxCount;  
}
```

캐릭터 클래스는 기본적으로 최대 점프 횟수를 변수로 가지고 있으며, 이를 변경함으로서 여러번 점프 할 수 있다.

그러므로 간단하게 어빌리트를 획득 시 변수를 지정하여 현재 캐릭터의 최대 점프 횟수를 변경하도록 구현하였다.

![[Character_DoubleJump.png]]

```C++
void UKYAnimInstance::UpdateCharacterState()  
{  
    ...
    bIsJumping = bIsFalling && WorldVelocity.Z > 0.f;  
    bIsDoubleJump = Owner->JumpCurrentCount > 1 && bIsJumping;  
    bIsCombat = Owner->GetIsCombat();  
    TimeToJumpApex = bIsJumping ? -WorldVelocity.Z / CharacterMovementComponent->GetGravityZ() : 0.f;  
	...
}
```

애니메이션은 캐릭터의 현재 점프 카운트가 1보다 큰지 확인하여 더블 점프 시퀀스로 트랜지션 될 수 있게 구성하였다.
